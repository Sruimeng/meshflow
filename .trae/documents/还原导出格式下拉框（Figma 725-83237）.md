**目标与范围**
- 实现与 Figma 规范一致的导出格式下拉框组件（正常/悬停/点击/禁用）
- 组件包含：当前选项显示、点击展开/收起面板、全部选项列表、状态提示与无嵌套交互结构
- 响应式适配，移动端 44px 触控高；桌面端匹配设计间距与圆角

**设计要点（来源节点 725-83237）**
- 尺寸：高度 44px；圆角 8px；水平内边距 12–16px
- 配色：背景半透明深色；边框 1px（透明白）；文字主色 #FAFAFA，副文案 #BDBDBD
- 图标：caret-down，大小 16px，与文字间距 8px
- 悬停：背景提高亮（rgba(255,255,255,0.12)）；边框稍强化
- 点击（active）：背景 rgba(255,255,255,0.16)
- 禁用：整体 50% 透明，指针不可用

**HTML 结构（demo/html/single.html）**
- 在现有 `.format-select` 区域替换 `<select>` 为自定义组件：
  - 容器：`<div class="format-select" id="J-format-combobox" role="combobox" aria-expanded="false" aria-controls="J-format-list" aria-haspopup="listbox">`
  - 触发按钮：`<button id="J-format-button" type="button" aria-labelledby="J-format-label">`
    - 文本：`<span id="J-format-label">Select Target Format</span>` 与当前选项 `<span id="J-format-value">GLB</span>`
    - 图标：内联 SVG caret-down
  - 面板：`<div id="J-format-list" role="listbox" aria-activedescendant="...">`
    - 选项：若干 `<div role="option" data-value="glb" aria-selected="true">GLB</div>` 等
- 旁路：保留一个隐藏 `<select id="J-format-native">` 同步值，供键盘/无 JS 回退或表单使用

**CSS 样式（demo/html/single.html）**
- 基础样式：
  - `.format-select`：display inline-flex；height 44px；border 1px solid rgba(255,255,255,0.12)；border-radius 8px；background rgba(255,255,255,0.06)
  - `.format-button`：flex；padding: 12px 16px；gap: 8px；color #FAFAFA；font 14px/24px
  - `.format-caret`：width/height 16px；填充 #FAFAFA
- 状态样式：
  - `:hover` 强化背景/边框；`:active` 深色；`:focus-visible` 自定义外环（不嵌套按钮）
  - `.is-disabled` 应用透明与 pointer-events: none
- 面板样式：
  - 绝对定位在容器下方；宽度与按钮等宽；圆角 8px；阴影；背景 rgba(0,0,0,0.9)
  - 选项项：高度 36–40px；左右内边距 12px；悬停背景高亮；选中项显示勾选或粗体
- 响应式：
  - 容器宽度随父容器自适应；在窄屏（<480px）缩小左右内边距为 12px；触控高维持 44px

**JS 交互（demo/src/single.ts）**
- 管理状态：`expanded`、`selectedValue`
- 事件：
  - 点击按钮：切换展开/收起（aria-expanded）
  - 点击选项：设置选中、更新显示、收起、同步隐藏 `<select>` 值与现有转换逻辑变量
  - 悬停：add/remove `.hover`；关闭区域外点击收起
  - 键盘：Enter/Space 打开；Escape 关闭；ArrowUp/Down 在选项间移动；Enter 选择
- 与现有转换集成：
  - 将 `fmt = formatSelect.value` 改为读取组件 `selectedValue`（并同步到 `#J-format-native`）
  - 保持转换按钮逻辑不变

**状态定义**
- 正常：初始显示默认值 GLB
- 悬停：背景/边框高亮
- 激活（面板展开）：按钮 active 样式、`aria-expanded=true`
- 选中：选项 aria-selected=true；按钮显示当前值
- 禁用：禁用整个组件（置 class `.is-disabled` 与 `aria-disabled=true`）
- 聚焦：`focus-visible` 外环

**图标处理**
- caret-down 采用内联 SVG，填充色与文本色一致；大小 16px；与文本间距 8px

**可访问性**
- 使用 ARIA combobox/listbox 角色与属性；选项使用 `role=option` 与 `aria-selected`
- Keyboard 支持完整导航；点击外部收起
- 避免交互嵌套：按钮是唯一触发元素，面板与选项非嵌套按钮

**交付物**
- HTML/CSS：插入到 `demo/html/single.html` 的组件标记与样式片段
- JS：在 `demo/src/single.ts` 添加组件逻辑，替换原生 `<select>` 的使用
- 响应式适配：CSS 媒体查询与弹性布局
- 状态样式：正常/悬停/点击/禁用/聚焦全覆盖
